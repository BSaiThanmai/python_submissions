1. integer:

2. FLOAT : 
--> It consists of numbers from +ve infinity to -ve infinity including zero with DECIMAL POINT.

->height = 6.2
->weight = 70.5

2. NON-PRIMITIVE DATATYPES (MULTI-VALUED DATATYPES):
--> These are also known as "COLLECTION DATATYPES".
--> These are used to store multiple values in a single variable.
--> we have 5 types:
1. STRING
2. LIST
3. TUPLE
4. SET 
5. DICTIONARY


1. STRING :
--> It is the collection of single or multiple characters enclosed between 
single quotes('') or double quotes("") or triple quotes(''' ''').

syntax : var_name = "value"
          var_name ='value'
         var_name = '''   
                         .......
                    '''  
properties of string :
--> String is immutable datatype. 
--> string is ordered collection.
--> String supports indexing. 

Indexing : providing sub-addresses to the each charater or element in a collection.

2. LIST :
--> It is the collection of HOMOGENOUS(same datatype) or HETEROGENOUS (Different datatypes)elemets enclosed between square brackets[] and elements are separated with comma(,).

syntax : var_name = [ ele1,ele2,....,ele-n]
         a=[20,30,50]
         l=["hello",20,60.89,True, 4+6j]

properties :
--> List is mutable datatype.
--> List is ordered collection.
--> It supports indexing.
-->default value for string is ''.


3. TUPLE:
--> It is the collection of homogenous and heterogenous elements enclosed with or without paranthesis and elements are separated by comma(,).

syntax : var_name = (ele1,ele2,...ele-n)
         var_name = ele1,el2,..ele-n
examples : t=(34,"hello",False,90.8)
           t = 34,"hello",False,90.8

properties of tuple :
-->Tuple is immutable datatype.
-->Tuple is ordered collection.
-->It supports indexing.
-->Tuple will allow duplicate values.
-->tuples are faster than lists.
